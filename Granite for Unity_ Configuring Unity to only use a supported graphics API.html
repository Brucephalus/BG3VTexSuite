Configuring Unity to only use a supported graphics API

Granite only supports DirectX 11 on Windows and OpenGL ES3 on android. This page goes into more detail on how to configure your Unity project so it can only run using one of the supported Graphics API's. As other rendering APIs are not supported this step is mandatory. Even if your game alays seems to run on DirectX 11/OpenGL ES3 during testing, on some user system configurations Unity may occasionaly select another API, even if the system should support DirectX 11.

  1. Open the player settings inside your Unity project: Edit > Project Settings > Player
  2. In the Other settings foldout, uncheck 'Auto Graphics API for Windows'
  3. Remove all rendering APIs except Direct3D11

Adapting the android settings works in the same way as described for the windows settings. Simply select the android tab and adjust the settings as mentioned above.
