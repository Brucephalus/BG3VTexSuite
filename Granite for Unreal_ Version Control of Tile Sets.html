Recommendations

If you are checking in changes into a version control system, we recommend you check in the following files:

  • *.grproj project file
  • *.grsln solution file (when using Tile Set Studio)
  • source texture files
  • *.gtex files

You can safely ignore:

  • *.log files
  • *.lock files
  • *.bak files
  • GrBuildState folder

Users who have access to the project file and source textures are capable of making changes to a project's Stacked Textures, build GTex and subsequent GTS files. Users who only have access to GTex files, can only build the GTS file. They cannot change any Stacked Texture properties.

Note
Building GTex files takes considerable time. Contrary, building GTS files from GTex files is a relative quick process. Therefore, we recommend you check in the GTex files.

Optionally, you can check in these files as well, but keep in mind they can have a substantial file size:

  • *.gts and *.gtp Tile Set files
Note
Although the tools allow you to build a GTS file from only the GTex files, Graphine recommends to check in your source texture files as well. This allows you to change properties of your Stacked Textures and recreate your GTex files. We do not guarantee the GTex format to be stable over different release versions, so current GTex files could be incompatible with future tools. Please keep your source textures so you can rebuild your GTex files.

Git

We recommend using the following gitignore rules:

# Temp folder
**/GrBuildState
# Temp and backup files
**/*.log
**/*.lock
**/*.bak
#Optionally, Tile Set files
**/*.gts
**/*.gtp

Perforce

In order to edit and build a solution or project, be sure to check out the following files:

  • grsln solution file
  • grproj project file(s)
  • any gtex files you will be changing

When not checked out, Perforce locks these files, and you will get an error in Tile Set Studio stating that the project files are not accessible.

We recommend using the following p4ignore rules:

# Ignore .p4ignore files
.p4ignore
# Temp folder
**/GrBuildState/
# Temp and backup files
**/*.log
**/*.lock
**/*.bak
#Optionally, Tile Set files
**/*.gts
**/*.gtp
